# JSX Viewer

A sleek, real-time React component previewer and comparison tool designed for developers and designers.

## Features

- **Real-time Preview**: Instantly see your JSX/TSX code come to life in the canvas.
- **Multi-Layout Support**:
  - **Single View**: Focus on one file with split-screen (code + preview) or canvas-only modes.
  - **Compare Mode**: Side-by-side comparison of 2 or 3 components at once.
- **Responsive Viewport Toggles**: Quickly test your designs on **Desktop**, **Tablet (768px)**, and **Mobile (375px)**.
- **Intuitive Zoom**: Use `Ctrl + Scroll` to zoom the canvas for pixel-perfect inspection without affecting the UI toolbars.
- **Seamless UI**: A minimalist, borderless canvas design that puts your work front and center.
- **File Management**:
  - **Drag & Drop**: Easily add files by dragging them into the workspace.
  - **Recent Files**: Quick access to your history of edited files.
  - **Download**: Export your active file with a single click.
- **Dark Mode**: Fully themed for comfortable coding in any environment.
- **Persistent Settings**: Your zoom level, viewport mode, and layout preferences are automatically saved.

## Getting Started

1. **Open a File**: Drag a `.jsx`, `.tsx`, or `.js` file into the app or use the "Add File" button in the tabs bar.
2. **Edit & Preview**: Use the split view to see changes in real-time as you refine your component.
3. **Compare**: Switch to 2-column or 3-column mode to compare different versions or related components.
4. **Test Responsiveness**: Use the viewport toggles in the header to ensure your UI looks great on all devices.

## Tech Stack

- **React 18**
- **Tailwind CSS**
- **Lucide React** (Icons)
- **Motion** (Animations)
- **Monaco Editor** (via CodeViewer)
